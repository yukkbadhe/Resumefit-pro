# ATS Resume Checker

An AI-powered web application that analyzes resumes against job descriptions and calculates an ATS compatibility score.

## Features

- Resume keyword matching
- Missing keyword detection
- ATS score calculation
- Resume section analysis
- Role prediction
- Suggestions for improvement
- Dark / Light mode
- Resume file upload support

## Tech Stack

- HTML
- CSS
- JavaScript

## How It Works

1. Paste your resume text
2. Paste the job description
3. Click **Analyze**
4. Get:
   - ATS Score
   - Matched Keywords
   - Missing Keywords
   - Improvement Suggestions
